    337 грн

    Опис

    Модулятор Borofone BC43 Flash – це простий у використанні трансмітер, який допоможе розширити мультимедійні можливості практично будь-якого автомобіля. Модель призначена для підключення до роз'єму прикурювача та після налаштування на одну зі штатним FM-приймачем хвилю дозволить без проводів відтворювати музику з носіїв інформації або зі смартфона через акустику автомобіля. Завдяки оригінальному дизайну з глянсовими далями, округленими гранями та матовою поверхнею, аксесуар лаконічно впишеться в салон, ставши його невід'ємним елементом. Для його виготовлення використовуються сучасні електронні комплектуючі та міцний, стійкий до навантажень пластик. На верхній панелі пристрою знаходяться зручні кнопки для налаштування та керування відтворенням, яскравий LED дисплей, що відображає поточну FM-станцію, USB роз'єм для заряджання гаджетів за протоколами QC, FCP та AFC з потужністю до 18Вт та USB порт для флешок із вихідною силою струму 1А. Також модель забезпечена слотом для карти пам'яті та модулем Bluetooth V5.0.
